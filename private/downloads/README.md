Lege hier pro Bundle einen eigenen Ordner an:

private/downloads/<bundle-id>/

Beispiel:

private/downloads/juniper-bundle-desktop-mobile/
  - presets.zip
  - installation-guide.pdf

Oder mehrere Einzeldateien:

private/downloads/juniper-bundle-desktop-mobile/
  - Juniper-Desktop.xmp
  - Juniper-Mobile.dng
  - Installation.pdf

Der Download wird nicht öffentlich aus dem Webroot ausgeliefert.
Die Academy-Auslieferung erzeugt nach Stripe-Zahlung einen zeitlich begrenzten Link auf den privaten Bundle-Ordner bzw. dessen ZIP-Archiv.
